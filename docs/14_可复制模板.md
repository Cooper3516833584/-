# 14 可复制模板

本文件提供项目初始化时可直接复制的模板。具体 Prompt 可以以后继续精修，但字段结构不要随意改。

## 1. agents/00_selector_agent.txt

```text
---
agent_id: selector
name: Agent选择器
enabled: true
kind: selector
priority: 100
model:
  name: default
  temperature: 0.1
  timeout_seconds: 60
  max_retries: 2
output_schema: SelectorResult
---

# Role
你是高校学生媒体智能审稿系统中的 Agent 选择器。

# Task
你只负责根据稿件内容、稿件元信息、初步规则判断结果和可用 Agent Catalog，选择本次审稿应该启用的审稿 Agent。

# Boundary
- 不直接审稿。
- 不输出具体修改意见。
- 不编造不存在的 Agent。
- 不选择 disabled Agent。
- 只输出符合 SelectorResult 的 JSON。
```

## 7. cases/case_privacy_photo.txt

```text
---
case_id: case_privacy_photo
title: 活动照片未确认授权引发投诉
case_type: privacy
severity: high
tags:
  - privacy
  - photo
  - authorization
related_agent_ids:
  - privacy_reviewer
---

【事件概述】
某活动报道使用了可清晰识别学生个人身份的近景照片，发布后当事人认为未经许可，要求删除。

【引发问题】
当事人认为照片发布缺少授权确认，且照片场景与表情不适合公开传播。

【复盘结论】
涉及可识别个人形象的图片，应确认授权或选择远景、背影、模糊处理等方式。

【审稿提醒】
检查稿件中是否出现照片、截图、姓名、班级、宿舍等可识别个人信息。
```

## 8. rules/student_media_review_rules.txt

```text
---
rule_id: student_media_review_rules
title: 学生媒体审稿基本规范
rule_type: internal_rule
authority_level: medium
tags:
  - review
  - privacy
  - factuality
---

1. 稿件中涉及具体个人、组织、活动方时，应保证事实准确、表述克制。
2. 涉及采访引语时，应避免断章取义。
3. 涉及个人信息、照片、聊天记录时，应确认授权或做必要匿名处理。
4. 标题、封面、摘要不得夸大事实，不得制造群体对立。
```
