# 15 配置项与环境变量

## 1. ReviewSettings

文件：

```text
src/review_tool/config.py
```

实现：

```python
from pydantic import BaseModel, Field

class ReviewSettings(BaseModel):
    llm_provider: str = "openai"
    default_model: str = "default"
    default_temperature: float = 0.2

    always_include_base_agents: bool = True
    base_agent_ids: list[str] = Field(default_factory=lambda: [
        "fact_checker",
        "risk_reviewer",
        "audience_reviewer",
        "format_reviewer",
        "privacy_reviewer",
    ])
    critical_agent_ids: list[str] = Field(default_factory=lambda: [
        "risk_reviewer",
        "privacy_reviewer",
    ])

    max_selected_agents: int = 10
    max_concurrent_agents: int = 4

    retrieval_top_k: int = 16
    max_context_chunks_per_agent: int = 8
    max_segment_chars_for_prompt: int = 12000

    debug: bool = False
    mock_mode: bool = False
```

## 2. .env.example

```text
LLM_PROVIDER=openai
OPENAI_API_KEY=
OPENAI_BASE_URL=
OPENAI_MODEL=

REVIEW_TOOL_DEBUG=false
REVIEW_TOOL_MOCK_MODE=false
```

## 3. 配置优先级

```text
CLI 参数 > 环境变量 > 默认 ReviewSettings
```

## 4. 模型名称处理

如果 AgentConfig 中：

```yaml
model:
  name: default
```

则使用 settings.default_model 或环境变量 `OPENAI_MODEL`。

如果 AgentConfig 中指定具体模型名，则使用 Agent 自己的配置。

## 5. Debug 模式

debug=True 时保存完整 prompt 和 raw output。

debug=False 时只保存必要 JSON 和报告。

## 6. Mock 模式

mock_mode=True 时不调用外部 LLM。

用途：

```text
单元测试
配置验证
无 API key 环境跑通流程
```
