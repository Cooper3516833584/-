# 07 审稿引擎主流程

## 1. 核心入口

文件：

```text
src/review_tool/engine.py
```

核心类：

```python
class ReviewEngine:
    def __init__(self, project_root: Path, settings: ReviewSettings): ...
    async def review_file(self, article_path: Path, output_dir: Path) -> ReviewRunResult: ...
    async def review_article(self, article: ArticleInput, output_dir: Path) -> ReviewRunResult: ...
```

CLI 和 Web API 都调用这个类。

## 2. ReviewEngine 初始化

初始化时执行：

```text
1. 读取 settings。
2. 加载 agents/*.txt。
3. 构建 AgentRegistry。
4. 加载知识库 txt。
5. 构建 KnowledgeRetriever。
6. 初始化 LLMClient。
```

伪代码：

```python
class ReviewEngine:
    def __init__(self, project_root: Path, settings: ReviewSettings):
        self.project_root = project_root
        self.settings = settings
        self.agent_configs = load_agent_configs(project_root / "agents")
        self.registry = AgentRegistry(self.agent_configs)
        self.documents = load_corpus(project_root)
        self.chunks = chunk_documents(self.documents)
        self.retriever = KnowledgeRetriever(self.chunks)
        self.llm_client = build_llm_client(settings)
```

## 3. review_file 流程

```python
async def review_file(self, article_path: Path, output_dir: Path) -> ReviewRunResult:
    article = load_article_file(article_path)
    article.source_path = str(article_path)
    return await self.review_article(article, output_dir)
```

## 4. review_article 流程

完整步骤：

```text
1. 生成 run_id。
2. 创建 output/{run_id}/。
3. 保存 article_snapshot.md。
4. 预处理稿件，得到 segments。
5. 构建 deterministic_hints。
6. 运行 Selector Agent。
7. 校验 SelectorResult，得到 selected_agent_ids。
8. 构建检索 query。
9. 检索知识库上下文。
10. 并行运行 selected_agent_ids 对应的审稿 Agent。
11. 校验和修复 Agent 输出。
12. 合并所有 Agent findings。
13. 风险评分和仲裁分层。
14. 生成 final_report。
15. 写 report.json、report.md、debug.log。
16. 返回 ReviewRunResult。
```

伪代码：

```python
async def review_article(self, article: ArticleInput, output_dir: Path) -> ReviewRunResult:
    run_id = make_run_id()
    run_dir = output_dir / run_id
    run_dir.mkdir(parents=True, exist_ok=True)

    save_article_snapshot(article, run_dir)

    segments = preprocess_article(article)
    hints = build_deterministic_hints(article, segments)

    selector_result, selector_debug = await run_selector(
        article=article,
        segments=segments,
        hints=hints,
        registry=self.registry,
        llm_client=self.llm_client,
        settings=self.settings,
    )

    selected_agent_ids = finalize_selected_agents(
        selector_result=selector_result,
        article=article,
        hints=hints,
        registry=self.registry,
        settings=self.settings,
    )

    retrieval_query = build_retrieval_query(article, segments, selector_result, hints)
    retrieved_context = self.retriever.retrieve(
        query=retrieval_query.query_text,
        tags=retrieval_query.tags,
        preferred_sources=retrieval_query.preferred_sources,
        top_k=self.settings.retrieval_top_k,
    )

    agent_results = await run_review_agents(
        agent_ids=selected_agent_ids,
        registry=self.registry,
        article=article,
        segments=segments,
        retrieved_context=retrieved_context,
        llm_client=self.llm_client,
        settings=self.settings,
    )

    validated_results = validate_agent_results(agent_results)
    merged_findings = merge_findings(validated_results)
    final_report = build_final_report(
        article=article,
        selector_result=selector_result,
        selected_agent_ids=selected_agent_ids,
        retrieved_context=retrieved_context,
        agent_results=validated_results,
        merged_findings=merged_findings,
    )

    result = ReviewRunResult(...)
    export_json(result, run_dir / "report.json")
    export_markdown(result, run_dir / "report.md")
    return result
```

## 5. run_id 规则

文件：

```text
src/review_tool/review/run_id.py
```

实现：

```python
from datetime import datetime
import secrets

def make_run_id() -> str:
    ts = datetime.now().strftime("%Y%m%d_%H%M%S")
    suffix = secrets.token_hex(3)
    return f"{ts}_{suffix}"
```

## 7. Agent 执行器

文件：

```text
src/review_tool/review/executor.py
```

接口：

```python
async def run_review_agents(
    agent_ids: list[str],
    registry: AgentRegistry,
    article: ArticleInput,
    segments: list[ArticleSegment],
    retrieved_context: list[KnowledgeChunk],
    llm_client: LLMClient,
    settings: ReviewSettings,
) -> list[AgentResult]:
    ...
```

要求：

```text
支持 asyncio 并行。
每个 Agent 独立 try/except。
单个 Agent 失败不导致整次审稿失败。
按 priority 排序，但并行执行。
结果回收后按 priority 排序。
```

## 8. Debug 保存

每次运行至少保存：

```text
output/{run_id}/selector_result.json
output/{run_id}/agent_results.json
output/{run_id}/retrieved_context.json
output/{run_id}/report.json
output/{run_id}/report.md
```

如果 settings.debug=True，额外保存：

```text
output/{run_id}/debug/
  selector_prompt.txt
  selector_raw_output.txt
  agent_risk_reviewer_prompt.txt
  agent_risk_reviewer_raw_output.txt
```

注意：debug 文件可能包含未发布稿件和内部材料，不建议提交到 Git。

## 9. 错误收集

不要直接 print 错误。统一记录：

```python
class ReviewError(BaseModel):
    stage: str
    error_type: str
    message: str
    detail: dict = Field(default_factory=dict)
```

ReviewRunResult.errors 存放所有错误。

## 10. 中止条件

只有以下情况中止整次审稿：

```text
稿件正文为空。
agents/ 目录不存在。
Selector Agent 配置不存在。
所有基础 Agent 都不存在。
LLM API 配置完全不可用，并且没有 mock 模式。
```

其它情况都应降级继续。
